# No Peek Cloud
Latest **release, with binary files**, can be found [here](https://github.com/HebronNor/NoPeekCloud/releases).

## What is No Peek Cloud?
No Peek Cloud let's you select folder on your local computer that you want to encrypt, when combined with a cloud service this provides a secure backup solution.
Source folders can be scanned periodically or every time a change is detected. 7-Zip is used for compression and encryption.

## Requirements
* Windows 7
* .NET Framework 4.5
* 7-Zip

## Command-line arguments
* Open application minimized: `NoPeekCloud.exe /Min`
* Syncronize when started: `NoPeekCloud.exe /Sync`
* Syncronize and exit when started: `NoPeekCloud.exe /SyncAndExit`
